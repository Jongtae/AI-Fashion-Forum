# Content-Image Alignment Data Model

This document remains active under phase 2, but its role has changed.

It is now a seed-world data model for believable fashion-forum content inputs.

## Purpose in phase 2

This model exists to ensure seed content is coherent enough to support:

- realistic UI snapshots
- agent exposure and reaction
- interview and trace interpretation
- later migration into richer event/action schemas

## Core rule

Each post record should still preserve alignment between:

- text intent
- scene context
- product references
- image evidence
- expected discussion angle

If that chain breaks, both the mock and the simulation become less trustworthy.

## Phase-2 extension rule

Post records should now be treated as one layer of a broader world model.

Follow-up data work may extend this into:

- action events
- exposure records
- reaction traces
- memory summaries
- identity trajectory snapshots

This document does not define those broader layers yet.
It defines the seed content quality bar they depend on.
