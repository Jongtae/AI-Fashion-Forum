# Seed-World UI: Historical Lessons

Phase 2 이전 UI 관련 의사결정에서 남은 교훈. 시드월드 표면 작업 시 회귀 방지 기준으로 참조.

---

## Outfit Preview Exposure

포럼 메인 경험에 실험적 미리보기 레이어가 과도하게 침투한 사례.

- 실험적 프리뷰 레이어는 메인 포럼 경험을 지배해서는 안 됨
- 현실감 보조 기능은 소셜/토론 가독성에 종속되어야 함

---

## Outfit Preview Privacy Variation

동일한 프라이버시 처리가 반복된 시드 에셋 문제.

- 반복된 템플릿 프라이버시 처리는 현실감을 약화시킴
- 에셋 다양성은 UI 신뢰도와 시뮬레이션 행동 모두에 영향을 줌

---

## Fallback Panels (Text-First)

에셋 누락 시 시스템 설명 UI가 포럼 세계를 덮어쓴 사례.

- 누락된 에셋 처리는 조용하게 유지해야 함
- 시스템 설명 UI가 커뮤니티 네이티브 포럼을 압도해서는 안 됨
- 시드월드 표면이 커뮤니티보다 시스템 설명에 가까워질 때 이 원칙을 회귀 가드로 사용
