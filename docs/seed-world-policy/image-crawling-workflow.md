# Image Crawling Workflow

This workflow remains active under phase 2 as a seed-material acquisition pipeline.

## Role in phase 2

Its job is to acquire and package assets that can be used in:

- the visible mock
- scenario packs
- seed-world datasets

It does not define final product behavior by itself.

## Core rule

Downloaded imagery is only a candidate input.
It becomes usable only after realism and alignment review confirm that it will not weaken the seed world or mislead later simulation analysis.
