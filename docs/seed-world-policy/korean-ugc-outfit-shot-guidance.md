# Korean UGC Outfit Shot Guidance

This document remains active under phase 2 as a seed-world realism guide.

## Role in phase 2

Believable Korean outfit imagery is still important because it shapes:

- what kinds of judgments agents can make
- whether the world reads as a real fashion forum
- whether interview and trace analysis stay grounded

## Core rule

Outfit imagery should still read like everyday Korean mobile UGC rather than campaign polish.

Under phase 2, this matters not only for UI credibility but also for simulation validity.
